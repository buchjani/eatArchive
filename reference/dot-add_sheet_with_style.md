# Format an excel sheet

Format an excel sheet

## Usage

``` r
.add_sheet_with_style(wb, sheet_name, df, autocomplete_values = NULL)
```

## Arguments

- wb:

  An excel workbook.

- sheet_name:

  Name of the excel sheet.

- df:

  A data.frame.

- autocomplete_values:

  A character vector of values which should be offered as autocomplete.
