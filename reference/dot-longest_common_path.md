# Find the longest common path in a vector of paths

Find the longest common path in a vector of paths

## Usage

``` r
.longest_common_path(vec_paths)
```

## Arguments

- vec_paths:

  A character vector of paths.

## Examples

``` r
if (FALSE) { # \dontrun{
paths <- c(
"c:/a/b/c/d",
"c:/a/b/e/f",
"c:/a/b/g/h",
"c:/a/i/j/d"
)
.longest_common_path(paths)
} # }
```
